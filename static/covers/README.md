# Book Cover Images

Please add book cover images to this folder:

1. **harry_potter.jpg** - Harry Potter and the Philosopher's Stone cover
2. **chronicles_of_narnia.jpg** - Chronicles of Narnia cover
3. **the_hobbit.jpg** - The Hobbit cover

## Image Specifications:
- Format: JPG or PNG
- Recommended size: 400x600px (portrait orientation)
- File size: < 500KB each
- High quality images for best display

## Where to get images:
- Book cover databases (ensure copyright compliance)
- Create custom illustrations
- Use placeholder images from Unsplash/Pexels (with proper attribution)

## Note:
The application will work without images, but covers enhance the user experience on the frontend.
