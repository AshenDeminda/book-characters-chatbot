# Book Characters Chatbot

RAG application that lets users chat with book characters based on uploaded books.

## Features
- Upload books (PDF/TXT)
- Automatic character extraction
- Character-based chatbot with personality

## Tech Stack
- FastAPI
- LangChain
- PostgreSQL/SQLite
- Vector Database (ChromaDB/FAISS)
